| **Tool**       | **Function**                       | **Command**                                                                 | **Description**                                                                                                               |
|----------------|------------------------------------|-----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| **Nmap**        | Basic Port Scan                   | `nmap <target_ip>`                                                         | Scans the 1000 most common TCP ports.                                                                                       |
|                | Aggressive Scan                   | `nmap -A <target_ip>`                                                      | Enables OS detection, version detection, script scanning, and traceroute (noisy).                                          |
|                | Service Version Detection          | `nmap -sV <target_ip>`                                                    | Determines service/application versions running on open ports.                                                               |
|                | OS Detection                      | `nmap -O <target_ip>`                                                      | Attempts to determine the target's operating system.                                                                        |
|                | Scan All Ports                    | `nmap -p- <target_ip>`                                                    | Scans all 65535 TCP ports.                                                                                                  |
|                | Ping Scan (Host Discovery)        | `nmap -sn <target_ip_range>`                                              | Only checks if hosts are up, no port scan.                                                                                  |
|                | Run Specific Scripts              | `nmap --script=smb-enum-users <target_ip>`                               | Runs Nmap Scripting Engine (NSE) scripts, e.g., for SMB enumeration.                                                       |
|                | Output to File                    | `nmap -oA nmap_scan <target_ip>`                                         | Outputs results in all formats (normal, XML, grepable) with the specified name.                                             |
|                | Stealth Scan (SYN scan)          | `nmap -sS <target_ip>`                                                    | The default and fastest scan method, less likely to be logged than a full TCP connect scan.                                 |
| **Gobuster**    | Directory/File Brute-force        | `gobuster dir -u http://<target_ip> -w <wordlist_path>`                  | Scans for hidden directories and files using a specified wordlist.                                                           |
|                | Specify Extensions                 | `gobuster dir -u http://<target_ip> -w <wordlist> -x php,txt,html`      | Looks for files with the specified extensions.                                                                               |
|                | DNS Subdomain Brute-force         | `gobuster dns -d <domain_name> -w <wordlist_path>`                        | Finds subdomains using a wordlist.                                                                                           |
|                | Verbose Output                    | `gobuster dir -v ...`                                                    | Provides more detailed output.                                                                                               |
| **Nikto**      | Basic Web Scan                    | `nikto -h <target_ip or hostname>`                                        | Scans the default port (80/443) for common vulnerabilities and configuration issues.                                         |
|                | Specify Port                      | `nikto -h <target_ip> -p 8080`                                            | Scans a web server running on a non-standard port.                                                                          |
|                | Scan with SSL                     | `nikto -h <target_ip> -ssl`                                               | Forces Nikto to use SSL for the connection.                                                                                  |
|                | Output to File                    | `nikto -h <target_ip> -o report.txt -F txt`                               | Saves the report to a file in the specified format (txt, html, etc.).                                                       |
| **Netcat (nc)**| Simple TCP Connection Test        | `nc -nv <target_ip> <port>`                                               | Simple TCP connection test.                                                                                                  |
|                | Listen for Incoming Connections    | `nc -lvp 4444`                                                            | Listen on a specific port for incoming connections (useful for reverse shells).                                             |
| **Host/Dig**   | Simple DNS Lookup                 | `host <domain_name>`                                                      | Simple DNS lookup.                                                                                                          |
|                | Attempt Zone Transfer (AXFR)     | `dig axfr @<dns_server_ip> <domain_name>`                               | Attempt a zone transfer.                                                                                                     |
